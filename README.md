# lite
Is a general purpose programming language that aims at being:
+ __Readable__
+ __Fast__
+ __Safe__
+ __Small__
+ __Independent__

### Examples
````
# Assignment

foo_0: i1 128

````

### Features
+ Strong typing
+ High interoperability with C
+ Powerful looping
+ Deterministic scoped memory managment
+ SOA (Structure Of Arrays) memory layout by default
+ Automatic vectorization
